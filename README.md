# CodeAlpha Data Analytics Internship

# Task 4 - Sentiment Analysis

## Project Overview

This project focuses on Sentiment Analysis using Python. Customer reviews were analyzed to determine whether each review expressed a Positive, Negative, or Neutral sentiment.

The project demonstrates Natural Language Processing (NLP) techniques for preprocessing text data, analyzing sentiment, and visualizing the final results.

---

## Objectives

- Analyze customer reviews
- Clean and preprocess text
- Perform sentiment analysis
- Classify reviews into Positive, Negative, and Neutral categories
- Visualize sentiment distribution
- Generate business insights

---

## Dataset

Mobile Reviews Sentiment Dataset

---

## Tools & Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

---

## Project Workflow

1. Import dataset
2. Data cleaning
3. Text preprocessing
4. Sentiment classification
5. Visualization
6. Business insights

Successfully classified customer reviews into different sentiment categories and visualized the overall sentiment distribution.
